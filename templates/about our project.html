Welcome to our Image Captioning Website! This project was made for IEEE MegaProject Submission.

This project uses a deep learning image captioning model consisting of an encoder-decoder CNN-RNN architecture. It uses a pretrained ResNet50 model as the encoder CNN to extract visual features from input images. This encoder outputs a feature vector for each image. The decoder is an LSTM RNN that takes this encoded image feature vector along with an input sequence of tokens (word embeddings) representing the caption so far. As it outputs predictions for the next word at each timestep, this allows the model to generate the caption one word at a time.
